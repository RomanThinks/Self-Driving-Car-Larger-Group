Line Tracking Robot

The add-on feature implemented:

 - IR Proximity Sensor with accompanied Buzzer

The IR sensor is a KY-032 4ppin IR Infrared Obstacle Avoidance Sensor Module
The Buzzer is a 2-5kHz Passive Buzzer Module (YL-44)

Both screenshots attached demonstrates the new configuration of sensors
facing the black line instead of outside the line.

Two videos demo the functionality of the robot.

 - One demos the new sensor configuration on the curved track. The attached pictures
   is the configurations that sensors were at when recording the session.

 - The other video demos the IR Proximity Sensor + Buzzer with the color sensor
   on a straight line.
